GtNotificationsService

Сервис для создания и управления уведомлениями. Уведомления отображаются в верхней части экрана и автоматически скрываются через заданное время.

Подключение

Сервис GtNotificationsService предоставляется в корневом инжекторе и готов к использованию. Для создания уведомления используйте метод add():

Уровни уведомлений

Уведомления поддерживают 4 уровня: info, success, warn, error. Уровень определяет цвет и иконку уведомления.

Длительность отображения

Параметр duration задаёт время отображения уведомления в секундах. По умолчанию — 10 секунд. Установите 0 для постоянного уведомления.

Кастомный компонент

В уведомление можно добавить кастомный компонент через параметр component.

Иконки

Для отображения иконки используйте параметры iconName (имя иконки) и iconClass (дополнительные CSS-классы). Иконка отображается перед заголовком уведомления.
